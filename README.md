# Alish's Calculator

It's my basic calculator project built with HTML, CSS, and JavaScript.

## Usage

1. Clone the repository: `git clone https://github.com/your-username/calculator.git`
2. Open the `index.html` file in your web browser.

## Features

- Addition
- Subtraction
- Multiplication
- Division
